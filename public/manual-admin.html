Modul Pendaftaran (Leads)

Bagian ini digunakan untuk memantau calon pelanggan yang mendaftar melalui portal publik atau referensi teknisi.

Kolom & Data Utama:
  • ID / Item Code: Kode unik referensi pendaftaran.
  • Status: Menunjukkan posisi leads (Waiting, Confirmed, dsb).
  • Pelanggan / Paket: Nama pendaftar dan paket internet yang dipilih.
  • Wilayah/Alamat: Lokasi fisik untuk survei dan instalasi.
  • Tanggal: Waktu entri data ke sistem.
Sistem Output:

Konversi pendaftaran ke Work Order akan mengubah status menjadi Confirmed, membuat record Installation Monitoring, dan mengirim notifikasi WhatsApp otomatis ke pelanggan.

Pendaftaran PSB (Input Manual)

Gunakan menu ini untuk mendaftarkan pelanggan baru secara langsung. Pastikan data berikut siap:

  • Foto KTP asli (untuk validasi NIK).
  • Koordinat lokasi (Lat/Long) untuk plotting area cakupan.
  • Informasi ODP terdekat untuk memastikan kelayakan teknis.

Detail Algoritma Point Engine
Logika Pembagian (Equal Split)

Sistem menggunakan rumus bagi rata untuk memastikan keadilan beban kerja.
Poin = Base_Point / Jumlah_Member

Tipe Pekerjaan & Poin Dasar
  • PSB (Pemasangan Baru): 2 Poin
  • REPAIR (Perbaikan): 1 Poin
  • MAINTENANCE: 1 Poin
  • UPGRADE: 1 Poin
100%

Integer Math Accuracy

No floating point math used.

Aturan Validasi Data (Constraints)

Format Pelanggan
  • Kode Pelanggan: YYMMXXXXXXX (11 digit).
  • KTP (NIK): Wajib 16 digit angka.
  • No. HP: Wajib diawali 08 atau 62 (9-13 digit).
Format Karyawan
  • Employee ID: 9 digit angka (YYYYMMXXX).
  • Email: Harus unik, digunakan untuk login sistem.
  • Target Poin: Default 1000/bulan (KPI).

Approval Payroll & Finance

Bendahara menggunakan menu ini untuk memproses gaji bulanan secara transparan dan akurat.

Fitur Utama & Kegunaan:
  • Seleksi Periode: Pilih bulan dan tahun gaji yang akan diproses.
  • Rekapitulasi Otomatis: Sistem menghitung Penghasilan Kotor, Potongan, dan Poin Teknisi secara real-time.
  • Status Pembayaran: Melacak status gaji dari Calculated (Dihitung) → Approved (Disetujui) → Paid (Lunas).
  • Ekspor CSV: Download laporan gaji dalam format Excel/CSV untuk kebutuhan pembukuan internal.
  • Slip Gaji: Cetak atau lihat slip gaji per individu teknisi dengan rincian poin pengerjaan.

Pengaturan WA (Fonnte)

Koneksi WhatsApp Gateway dikelola melalui menu Sistem Settings pada grup WhatsApp / Fonnte.

Lokasi Setting:

Buka menu System → Settings → Klik tab WhatsApp / Fonnte. Di sini Anda dapat memasukkan API TOKEN Fonnte.

Monitoring Status:

Status koneksi perangkat (Connected/Disconnected) dapat dilihat di dashboard notifikasi. Jika pesan tidak terkirim, pastikan perangkat WA Anda aktif dan terhubung.

Kamus Istilah & Menu (Glossary)
Modul Pendaftaran
Menu pusat untuk memantau calon pelanggan (Leads) yang masuk ke sistem.
Pendaftaran PSB
Menu untuk menginput data pendaftaran pelanggan baru secara manual oleh Admin.
Installation Monitoring
Log histori real-time yang mencatat setiap langkah pengerjaan teknisi di lapangan.
Point Engine
Sistem otomatis yang menghitung bonus teknisi berdasarkan penyelesaian Work Order.
Fonnte
Gateway API WhatsApp yang digunakan sistem untuk mengirim pesan otomatis secara instan ke pelanggan dan tim.

Bantuan Cepat (Troubleshooting)

Poin Tidak Muncul?

Pastikan status tiket sudah mencapai Closed. Poin hanya dihitung secara otomatis saat tiket ditutup secara resmi oleh admin/sistem.
